反码
```js
num1 = 77 = 0_77
num2 = 76 = 0_76
num1 - num2 = 77 + (-76) = 77 + (100-76) -100  = 77 + 34 -100 = 107 -100=>7
num1 - num2 = 正_77 + 负_10 = 正_77 + 负_90 = 负_1_67 => 正_67

num1 = 17 = 0_10
num2 = 47 = 0_47
num1 - num2 = 17 + (-47) = 17 + (100-47) =17+53 = 70 -100 = -30
num1 - num2 = 正_17 + 负_47 = 正_17 + (负_53) = 负_70 => 转原码 负_30
```
